# Module 06 - ASP.NET Core Web API

## Objective
To develop a RESTful Web API using ASP.NET Core and perform CRUD operations.

## Topics Covered
- ASP.NET Core Web API
- REST API
- HTTP Methods
- Controllers
- Routing
- Swagger
- Entity Framework Core
- CRUD Operations

## Concepts Learned

### Web API
A Web API allows applications to communicate over HTTP by exposing endpoints.

### REST API
REST (Representational State Transfer) is an architectural style used to build web services.

### HTTP Methods
- GET - Retrieve data
- POST - Add new data
- PUT - Update existing data
- DELETE - Remove data

### Controller
A Controller handles incoming HTTP requests and returns appropriate responses.

### Swagger
Swagger provides a user interface to test and document Web APIs.

### Entity Framework Core
EF Core is used to connect the Web API with SQL Server and perform database operations.

## Steps Performed
1. Created ASP.NET Core Web API project.
2. Installed Entity Framework Core packages.
3. Created Product model.
4. Configured AppDbContext.
5. Added SQL Server connection string.
6. Implemented CRUD API endpoints.
7. Tested APIs using Swagger.
