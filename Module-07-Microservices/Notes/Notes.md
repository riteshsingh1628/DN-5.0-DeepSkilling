# Module 07 - Microservices Notes

## What is a Microservice?

A microservice is a small, independent service that performs a specific business function.

## Advantages
- Independent deployment
- Easy maintenance
- Better scalability
- Fault isolation

## Components
- API
- Database
- Controller
- Model

## Learning Outcome
- Understood Microservices architecture.
- Built a Product Service using ASP.NET Core Web API.
