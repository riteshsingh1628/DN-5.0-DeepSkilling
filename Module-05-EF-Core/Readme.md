# Module 05 - Entity Framework Core

## Objective
This module demonstrates how to use Entity Framework Core with SQL Server LocalDB to perform CRUD (Create, Read, Update, Delete) operations using the Code First approach.

## Technologies Used
- C#
- .NET
- Entity Framework Core
- SQL Server LocalDB
- Visual Studio 2022

## Project Name
RetailInventorySystem

## Features
- Database creation using EF Core
- Product model creation
- SQL Server LocalDB connectivity
- Create Product
- Read Product
- Update Product
- Delete Product
