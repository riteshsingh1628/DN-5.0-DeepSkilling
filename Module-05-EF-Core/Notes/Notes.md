# Module 05 - Entity Framework Core
## Exercise: Understanding ORM with a Retail Inventory System
### Objective
To learn Entity Framework Core and perform CRUD operations using SQL Server LocalDB.
### Topics Covered
- Object Relational Mapping (ORM)
- Entity Framework Core
- DbContext
- DbSet
- SQL Server LocalDB
- Code First Approach
- CRUD Operations
### Concepts Learned
#### ORM
Object Relational Mapping (ORM) maps C# classes to database tables.
#### Entity Framework Core
Entity Framework Core is Microsoft's ORM framework that simplifies database operations.
#### DbContext
DbContext manages the connection between the application and the database.
#### DbSet
DbSet represents a table in the database.
#### CRUD Operations
- Create
- Read
- Update
- Delete

### Steps Performed
1. Created Console Application.
2. Installed Entity Framework Core packages.
3. Created Product model.
4. Created AppDbContext.
5. Connected SQL Server LocalDB.
6. Created database.
7. Performed Create, Read, Update and Delete operations.

- CRUD Operations Completed Successfully.
